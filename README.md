# Modelling_Matlab_20-sim
Tutaj będę umieszczał projekty które wykonywałem w ramach zajęć na uczelni agh związanych z zagadnieniem modelowania. W tym celu używałem środowiska Matlab i toolboxów jak Simulink. Na uczelni w Holandii miałem również styczność z modelowaniem w tym na przestrzeni nauki wykonywałem projekt z użyciem Relbota w środowisku 20 sim.
